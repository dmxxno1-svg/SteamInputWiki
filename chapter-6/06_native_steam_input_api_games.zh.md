# 第 06 章：原生 Steam Input API 游戏（Native SIAPI Games）

如 SIAPI 章节所述，部分开发者已在游戏中实现了 SIAPI，这提供了“原生”的 Steam Input 支持，从而能更好地支持 Steam 控制器等硬件。

原生实现的优势包括：

* 绑定动作而非物理按键
* 根据游戏上下文自动切换 Action Sets
* 游戏内按钮提示会与玩家所用控制器及其自定义配置保持一致
* 以及更多功能

但并非所有实现都优秀：有些开发者实现得很好，有些则留有缺陷。因此我们希望建立一个评分体系，帮助玩家在购买前了解某款游戏 SIAPI 实现的质量，也为开发者提供可参考的改进目标与示例。

本章将构建该评分体系并提供对开发者有帮助的资源与最佳实践建议。
