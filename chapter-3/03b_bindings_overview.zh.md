# 第 03b 章：绑定概览

本章将更深入地介绍你可以绑定到按钮的各种项目。

## 鼠标按键

- 左/右键点击
- 向上/向下滚动
- 中键点击
- 鼠标按钮 4 与 5
- 鼠标增量绑定（Mouse Delta Binding）——位于鼠标图形下方中间的 4 个箭头
  - 该绑定允许你设置一个按键，让鼠标光标沿某个方向微移一定量。
  - 选择该绑定后，会弹出对话框，允许你通过 X 轴和 Y 轴滑块配置方向与幅度。
  - 在 X 轴滑块上，负数向左移动光标，正数向右移动。
  - 在 Y 轴滑块上，正数向下移动光标，负数向上移动。

## 键盘按键

这些大多是“所见即所得”的键盘按键。

使用这些绑定时，游戏可能会假定你正在使用键盘和鼠标，从而显示相应的按钮提示。

缺失的键包括：

- F13–F24 键
- Fn（功能键）

如你想到其他遗漏的键，请补充到清单中。

## 手柄按键

注意：如果你在桌面配置中使用非 Xbox 控制器，这些按键可能不可用。

这些是 XInput（Xbox）命令。使用这些绑定时，游戏会认为你在使用 Xbox 控制器，因此会显示 Xbox 控制器提示/图标。例外情况是开发者通过 Steam Input API 查询了正在使用的控制器类型——这不需要完整实现 SIAPI。

- 绑定扳机时，会将扳机的完整值发送到游戏。
- 你可以通过选择摇杆方向的边缘来绑定摇杆方向。
  - 但这样会向游戏发送完整值。
- 通过选择摇杆的中心来绑定摇杆按下（Joystick Click）。

## SIAPI 游戏内动作（SIAPI In-Game Actions）

若游戏实现了 SIAPI，该游戏会提供预定义的绑定（Actions）供你分配。在此模式下，你不会看到鼠标、键盘或手柄按键，而是看到一组由游戏定义的动作网格。具体动作内容因游戏而异。

## Steam / Steam Input / 系统绑定

这些绑定用于与 Steam、Steam Input 或你的系统交互。下面按从左到右的顺序说明：

- 动作集或动作图层绑定（Action Set / Action Layer）——图标为 3 个叠放的圆圈带循环箭头
  - 仅在配置拥有多个动作集或存在动作图层时可见。
  - 动作集与图层将在第 04 章中详细介绍。
- 显示键盘（Show Keyboard）——图标像键盘
  - 打开 Steam 的屏幕键盘。
- 截图（Take Screenshot）——图标像相机
  - 触发 Steam 的截图并保存到你的截图库。
- 移动光标（Move Cursor）——图标像带圆圈的鼠标光标
  - 与鼠标增量绑定不同，此绑定将光标移动到屏幕上的特定位置。
- 切换放大镜（Toggle Magnifier）——图标像放大镜
  - 打开系统的放大镜辅助功能。
- 电源选项（Power Options）——图标像电源按钮
  - 该绑定包含多个功能，选择后会弹出下拉菜单供你选择：
    - 关闭控制器（Turn off Controller）——立即关机控制器
    - 退出应用（Exit Application）——强制退出当前游戏或焦点程序
    - 打开大屏模式（Open Big Picture）
    - 最小化大屏模式（Minimize Big Picture）
    - 退出大屏模式（Quit Big Picture）
    - 挂起主机 PC（Suspend Host PC）——Steam Link 功能：让主机进入睡眠
    - 关闭主机 PC（Shutdown Host PC）——Steam Link 功能：关闭主机
    - 重启主机 PC（Restart Host PC）——Steam Link 功能：重启主机
    - 触屏悬停 / 左键 / 右键 / 中键 / 本机触屏（Touchscreen Hover / Left Click / Right Click / Middle Click / Native）——行为依平台而异
- 设置灯光选项（Set Light Options）——图标像灯泡
  - 允许自定义控制器灯光，打开一个包含下拉菜单和两个滑块的对话框：
    - 下拉菜单在三种行为间选择：默认用户设置（General Controller 设置）、默认 XInput 插槽编号（用于显示玩家编号）、自定义设置（使用下方滑块）
    - 控制器灯光颜色滑块选择颜色（仅在控制器支持 RGB 时有效）
    - 控制器灯光饱和度滑块决定亮度。
- Steam 音乐绑定（Steam Music Bindings）——图标像音符
  - 与 Steam 的音乐播放器交互，选择后弹出对话框让你选择控制项：播放/暂停、下一曲、上一曲、音量增/减、静音。
- 空绑定（EMPTY Binding）——图标像中间带横线的圆
  - 此绑定不执行任何操作。
  - 在应用动作图层时若想让某个按键不执行任何操作，应使用空绑定（直接移除绑定由于 Steam Input 的限制可能无效）。
- 更改控制器玩家编号（Change Controller Player Number）——图标像带两个人像的控制器
  - 在本地多人游戏中，该绑定允许你选择当前玩家编号（1-16），避免来回传递控制器。
